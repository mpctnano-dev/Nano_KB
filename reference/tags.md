# Tags

Use tags lightly. Tags support discovery across page guides and references, but they do not replace navigation.

## Starting Tag Set

| Tag | Use For |
| --- | --- |
| `HTML` | Target HTML files and markup snippets |
| `CSS` | Selectors, visual classes, and style side effects |
| `JavaScript` | Script dependencies, dynamic behavior, and ids read by JS |
| `Forms` | Form fields, validation, submission, and category routing |
| `Navigation` | Header, footer, menu, active states, and links |
| `Equipment` | Equipment catalog, reservation, status, and instrument content |
| `Content Edit` | Small text, card, CTA, stat, and label changes |
| `Reference` | Reusable docs conventions |
