# Getting Started

Use this section to understand the maintenance model for the MPaCT public website and the MPaCT Docs Site.

## Maintenance Model

- The MPaCT public website is the user-facing site at `nano.nau.edu`.
- The MPaCT Docs Site explains how to update and maintain that public website.
- Existing page guides focus on exact files, selectors, page sections, and side effects.

## First Workflow

1. Search for the public page or selector you need.
2. Open the matching page guide.
3. Read the side effects before editing.
4. Make the smallest safe change.
5. Verify the public page behavior in a browser.
