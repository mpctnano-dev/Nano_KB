    1. Degree Program Rows¶

    Add a New Degree Program¶

    Each program is an <article> element inside the program-rows div. Copy one complete article block (from <article> to </article>) and paste it at the end, before the closing </div> of program-rows.

    Search for: <div class="program-rows">

    <!-- Paste before the closing </div> of program-rows -->
<article class="program-row reveal-on-scroll" id="your-program-id">
  <div class="program-row-image">
    <img src="Images/your-image.jpg" alt="Program description">
  </div>
  <div class="program-row-content">
    <div class="program-row-header">
      <h3 class="program-row-title">Program Name</h3>
      <span class="program-row-degree">Bachelor of Science</span>
    </div>
    <p class="program-row-desc">Description of the program...</p>
    <div class="program-row-features">
      <div class="program-row-feature">
        <!-- SVG icon goes here (copy from any existing feature) -->
        Feature Label
      </div>
      <!-- Add more features as needed -->
    </div>
    <a href="https://degree-search.nau.edu/..." target="_blank"
       class="program-row-link">
      View Degree Details
      <!-- Arrow SVG (copy from any existing program-row-link) -->
    </a>
  </div>
</article>

    Note

    The id attribute on the article (e.g. id="computer-engineering") is used by internal anchor links from other pages. Give it a short, hyphenated id that matches the program name.

    Edit a Program Title, Description, or Degree Badge¶

    Search for: class="program-row-title"

    1
2
3
    <h3 class="program-row-title">Computer Engineering</h3>
<span class="program-row-degree">Bachelor of Science</span>
<p class="program-row-desc">Edit description text here...</p>

    Edit a Feature Tag Inside a Program¶

    Each program has up to 4 feature items. Each has an SVG icon and a text label. To change the label, edit the text after the closing </svg> tag.

    Search for: class="program-row-feature"

    1
2
3
4
    <div class="program-row-feature">
  <svg ...>...</svg>
  Computer Architecture    <!-- Edit this text -->
</div>

    Note

    The SVG icons are inline. If you want a different icon, copy one from another feature in the file or use a different inline SVG path. Font Awesome is NOT used for these icons.

    Temporarily Hide a Program¶

    Search for: id="electrical" (or the id of the program)

    1
2
3
4
5
6
    <!-- Comment out the entire article: -->
<!--
<article class="program-row reveal-on-scroll" id="electrical">
  ...
</article>
-->

    Side Effect

    Program rows are block-level elements stacked vertically. Hiding one closes the gap cleanly - adjacent programs will move up. No CSS adjustment needed.

    2. Hero Stats Panel¶

    Edit a Stat Number or Label¶

    Search for: <div class="ml-hero__panel-stats">

    1
2
3
4
5
    <div class="ml-pstat__num">6</div>        <!-- the number -->
<div class="ml-pstat__label">Degree Tracks</div>  <!-- the label -->

<!-- With a suffix in lighter weight: -->
<div class="ml-pstat__num">100<em>%</em></div>

    Note

    The <em> tag renders the suffix character in a lighter style. Keep it for symbols like %, +, K. Remove for a plain number.

    Add or Remove a Hero Tag¶

    Search for: <div class="ml-hero__panel-tags">

  <!-- Add a new tag: -->
  <span class="ml-hero__panel-tag">Industry Mentors</span>
</div>

  </div>

    Note

    The second campus card uses campus-card-badge-gold for a gold badge variant. Keep that class if you want the gold color, or remove it for the default blue.
